# 🛒 E-Commerce Order Management System

## Overview
This project is a SQL-based database system designed to manage users, products, orders, and payments.

## Features
- Normalized schema
- Data insertion
- SQL queries for reporting
- Stored procedures
- Triggers for automation
- Indexing for performance

## Files
- Sql-01_schema.sql → Database schema
- Sql-02_data.sql → Sample data
- Sql-03_queries.sql → Queries & reports
- Sql-04_procedures.sql → Stored procedures
- Sql-05_triggers.sql → Trigger logic
- Sql-06_indexes.sql → Performance tuning

## How to Run
Execute files in order:
1. Schema
2. Data
3. Queries
4. Procedures
5. Triggers
6. Indexes

Tested on MySQL with sample data. All queries executed successfully.
